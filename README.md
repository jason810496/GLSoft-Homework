### Calculator
- Requirements:
    - number : `0-9` , `0.X`
    - Basic calculation : `+` , `-` , `*` , `\` , `%` , `+/-` , `=`
    - memery : `m+`,`m-`,`m` 
    - clear : `ac`
    - multi-theme
    - use none server side storage ( `localStorage` )
